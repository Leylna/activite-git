Git & GitHub -Partie2 Activité:

Tout d'abord créer un repository Git simple contenant:
	-Un fichier nommé "README.txt qui va présenter le projet.
	- Au moins 2 autre fichiers au choix(script/code/du texte d'un projet perso)
	- Au moins 4 commits distincts montrant l'ajout puisl a modification de certain fichiers.

Tous en commenant ces commits avec des commentaires utiles et non commit1 et commit2

Une fois le repository Git crée, envoyez-le sur le GitHub, puis copiez l'URL du repository et indiquez l'url de votre repository GitHub dans le document 
texte à télécharger à la suite de "L'URL de mon repository sur GitHub". Puis compressez ce document txt au format .zip et envoyez le.
